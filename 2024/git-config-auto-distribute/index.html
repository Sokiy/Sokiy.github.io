自动区分工作仓库和个人仓库的 Git 提交

通常，公司会自建 GitHub 或 GitLab 服务，而个人可能在 GitHub 官网上有自己的个人项目。然而，公司内部项目和个人项目的邮箱配置可能不同。当项目数量较多时，使用 git config --local 或 --global 进行设置就会变得相当麻烦。在这里，我将记录下我个人的解决方案，以便后续参考。
